# GELDWERT

**Unabhängige Tests & Empfehlungen** — eine partner-agnostische Affiliate-Review-Plattform.
Von Energie und Internet über Finanzen bis Reise. Das Partner-Netzwerk wird laufend ausgebaut.

## Inhalt

- `index.html` — eigenständige, gehostete Beispiel-Review (Check24-Produkte: Strom & Gas, DSL, C24 Bank, Mietwagen). Eine einzelne, in sich geschlossene HTML-Datei ohne Build-Schritt.

## Aufbau

- **Aesthetik:** editorial / Finanz-Magazin — warmes Papier, Tinten-Schwarz, ein Honig-Gold-Akzent.
- **Typografie:** Fraunces (Display), Newsreader (Fließtext), IBM Plex Mono (Daten/Labels) via Google Fonts.
- **Features:** Lesefortschritts-Balken, Sticky-Inhaltsverzeichnis mit Scroll-Spy, Produkt-Karten (Score, Pro/Contra, Eckdaten, CTA), Vergleichstabelle, FAQ-Akkordeons, Scroll-Reveals, Affiliate-Transparenzhinweis.
- **Technik:** reines HTML/CSS/JS, keine Abhängigkeiten, keine Build-Pipeline.

## Lokal ansehen

`index.html` einfach im Browser öffnen.

## Hosting

Per GitHub Pages bereitgestellt (Branch `main`, Root `/`).

## Affiliate-Hinweis

Die `href="#"`-Platzhalter in den CTA-Buttons durch die echten Affiliate-/Partner-Links ersetzen.
Affiliate-Links tragen `rel="sponsored nofollow noopener"`.
